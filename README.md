# Introducción a los Microcontroladores

Todas las practicas, examenes, extras, de la clase de Introducción a los Microcontroladores con el profesor Juan Pérez Pérez, de la Escuela Superior de Computo

Este repositorio contiene: 
  - Practica 1 a la 10
  - Examen
  - 
### Técnico

En este repositorio se usaron las siguientes tecnologias.

* [Atmel Studio](https://www.microchip.com/mplab/avr-support/atmel-studio-7) - IDE para el desarrollo en ensamblador.

### Instalación

Se requiere [git](https://git-scm.com/) para ejecutar los siguientes comandos en consola:

```sh
$ git clone https://github.com/IrvingGuerra/Introduccion-a-los-Microcontroladores
```
